# Landing Page Project

## Description

This project dynamically builds a navigation menu, enables smooth scrolling, and highlights sections in the viewport as you scroll through the page.

## Features

- Dynamic navigation bar generation based on sections in the HTML.
- Smooth scrolling to sections when navigation items are clicked.
- Highlights active section and corresponding navigation link.

## Instructions

1. Clone or download the repository.
2. Open the `index.html` file in any modern browser.
3. Enjoy the interactive and responsive landing page.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)

## Future Improvements

- Add animations when sections come into the viewport.
- Dynamically add or remove sections via JavaScript.
